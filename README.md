#  CarbonPulse

CarbonPulse is an AI-powered sustainability tracking platform built using Flutter and Firebase.
The app helps users monitor, analyze, and reduce their carbon footprint through smart insights, OCR bill scanning, AI recommendations, analytics, and gamified eco rewards.

---

#  Features

##  Firebase Authentication

* Secure login & signup
* Email verification
* User profile management

##  AI Bill Scanner

* Upload electricity bills
* OCR-based text extraction
* Detect electricity usage automatically

##  Carbon Analytics

* Real-time carbon tracking
* Dynamic charts and trends
* Monthly sustainability insights

##  CarbonPulse AI Assistant

* AI-powered eco recommendations
* Sustainability guidance
* Smart carbon reduction suggestions

##  Eco Rewards System

* Eco points & achievements
* Sustainability streaks
* Gamified user experience

##  AI Sustainability Reports

* Monthly eco reports
* Carbon summaries
* Personalized sustainability insights

##  UI

* Dark/Light mode
* Responsive Flutter UI
* Cross-platform support
